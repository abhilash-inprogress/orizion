# Orizion 🎙️

Built by Abhilash & Ananya

This is a small project we built to explore one simple question:

**Can we tell if a voice is real or AI-generated?**

You give it an audio file, and it tries to predict whether it's REAL or FAKE.

---

## What it does

- Takes an audio file as input  
- Converts everything into a consistent format  
- Extracts features from the sound  
- Uses a trained machine learning model  
- Outputs a prediction: REAL or FAKE  

---

## Tech used

- Python  
- Librosa  
- Scikit-learn  
- Streamlit  

---

## How to run

```bash
pip install -r requirements.txt
python scripts/train.py
streamlit run app.py
```

---

## Why we made this

AI voices are getting really good, and we wanted to understand what separates them from real human voices.

This project was mainly about experimenting and learning.
